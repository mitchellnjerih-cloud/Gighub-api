# GigHub Freelance Gigs API

**Student Admission Number:** C027-01-2287/2024

## Project Description

GigHub Freelance Gigs API is a RESTful API built using FastAPI. It allows clients to create, manage, search, update, and delete freelance gig listings. The API also supports filtering gigs by category and budget range, making it easier for freelancers to find suitable opportunities.

## Features

- View all available gigs
- Search gigs by title
- Filter gigs by category and budget
- View a single gig by ID
- Add a new gig
- Update a gig's budget or status
- Delete a gig

## Technologies Used

- Python 3
- FastAPI
- Uvicorn
- Pydantic

## Running the Project

Activate the virtual environment:

```powershell
.\venv\Scripts\Activate
```

Run the API:

```powershell
uvicorn main:app --reload
```

Open Swagger UI:

```
http://127.0.0.1:8000/docs
```
